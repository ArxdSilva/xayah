
# xayah

<img src="img/xayah.png" width="100">

xayah is a command line tool to help adding owners to APIs, in multiple development environment only once.
> requires: roadie2

### Usage:

> install
```shell
    $ go get github.com/arxdsilva/xayah
    $ cd arxdsilva/xayah
    $ go install
```

> run binary
```shell
    $ xayah
```
